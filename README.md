
Были использованы:
- Tornado
- Coffin
- Pillow
- South
- Redis (для хранения сессий)

---
Как всё это запускать?

Сначала нужно настроить settings.py и tornading.py. Установить всё, что написано выше, на вашу систему.

Запускается командой
`python tornading.py start`
на 8001 порту
